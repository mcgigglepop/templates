Create Multiple Items

Upload CSV

Fill out a CSV template and upload to create multiple items. You can view the status and edit the items later in your dashboard. Add any number of custom fields in the CSV file.

1. Download the Template

2. Title is required

3. Change "Custom Field" to the custom field name that you want (or delete it)

4. Change "Custom Value" to the custom value you want (or delete it)

5. Add as many Custom Field/Custom Value that you want

For each item, choose a Collection. If no collection is provided, the item will automatically be assigned to your default collection. If a collection is provided that does not exist, a private collection will be created and the item assigned to it.
